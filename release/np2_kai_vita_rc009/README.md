# NP2 Kai Vita RC009

Install `vita_np2_kai_fullcore_rc009.vpk` on PS Vita.

Recommended stable config:

```ini
VITA_PRESENT_SKIP=3
VITA_CORE_DRAWSKIP=false
VITA_AUDIO_LIGHT=false
VITA_SPEED_LIMIT=0
```

High-speed beta config:

```ini
VITA_PRESENT_SKIP=3
VITA_CORE_DRAWSKIP=true
VITA_AUDIO_LIGHT=false
VITA_SPEED_LIMIT=0
```

Docs:

- `release_notes_v009.md`
- `vita_config.md`
- `known_issues.md`

VPK files are not meant to be committed to Git. Attach them to a GitHub Release.
